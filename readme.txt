100% Free Google Recaptcha for Woocommerce Checkout - Add your Google Recaptcha API keys and done!

== Description ==

First - Go make some keys and add your website as a Domain here: https://www.google.com/recaptcha/admin and match options in screenshot.

Second - Install and active this plugin

Third - Go to plugin options page: /wp-admin/options-general.php?page=sudo_google-recaptcha

Finally - Add keys from first step and save changes.
